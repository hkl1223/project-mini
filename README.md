# 渝康码属性
|  属性   | 说明  |   类型   | 可选值  | 默认值 |
|  ----   | ----  |  ----   |  ----  |  ----  | 
| list  | 列表 |    Array[] |  -  |    []     |
| reportList  | 列表 |Array[] |  -  |    []   |
| serverList  | 列表 |Array[] |  -  |    []   |
| feverList  | 列表 |Array[] |  -  |    []   |

### 示例

1.引用：小程序在当前使用的json文件里面引入

     "yukangCode": "/components/yukangCode/yukangCode" 

2.使用：list传参

     <yukangCode list="{{yukangCodeList}}"></yukangCode>
